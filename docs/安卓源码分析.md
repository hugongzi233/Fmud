## 二、网络通信层（mudsocketd.java）

| 项目 | 内容 |
|------|------|
| 连接类型 | TCP Socket（`java.net.Socket`） |
| 读写 | `BufferedReader.readLine()` 读，`PrintWriter.println()` 发 |
| 字符编码 | UTF-8 |
| 心跳 | `socket.setKeepAlive(true)` |
| 重连 | `new_sock()` 异步创建新连接，`recon()` 触发 |
| 状态管理 | Application 单例，全局共享玩家信息 |
| 发送函数 | `mudsocketd.mudout(String str)` |
| 接收函数 | `mudmaind` 主循环 200ms 轮询 `mudin()` |
| 加密 | Socket 本身**明文**，无加密 |

## 三、登录认证流程（logind.java）

### 1. 版本检查
```
HTTP GET: https://mud1.foxmoe.top/mobi/vers.php
       ↓
响应: 版本号 | 服务器提醒文本
```

### 2. 登录/获取服务器列表
```
HTTP GET: https://mud1.foxmoe.top/mobi/loginto.php
          ?id={账号}&pass={密码}&key=MD5(id+pass+"AP4s3dF5")
          &page={页码}
       ↓
响应: $l#服务器1&IP&端口&key|服务器2&IP&端口&key|...|$l#最后一页
```

### 3. 注册
```
HTTP GET: https://mud1.foxmoe.top/mobi/reg.php
          ?id=&pass=&phone=&email=&key=MD5(id+pass+phone+"AP4s3dF5")
       ↓
响应: 注册成功 / 失败原因
```

### 4. 选服后建立 TCP 连接
```
mudsock.new_sock(IP, 端口)  →  TCP Socket 三次握手
       ↓
发送: 账号║密码║key║email
       ↓
等待: 版本验证成功 / 客户端非法 / 验证失败
       ↓
跳转 mudmaind.java
```

## 五、服务端→客户端协议（ 前缀）

按消息前 6 字符（`00X`）分发：

| 协议码 | 含义 | 处理函数 |
|--------|------|---------|
| `000` | 重连完成 | 发送 "l" |
| `001` | 隐藏蒙版/等待 | `mengban1.GONE` + `dengdaijindu1.GONE` |
| `002` | 更新标题 | `taketitle()` 重置界面按钮状态 |
| `003` | 出口/方向数据 | `takeexit()` 渲染方向按钮 |
| `004` | 剧情文字 | `story_text.setText(spantxt())` |
| `005` | 对象列表（玩家/NPC） | `takeobj()` 渲染左侧列表 |
| `006` | 自定义命令按钮 | `takebutton()` |
| `007` | 目标详情 | `setmubiaoinfo()` |
| `008` | 互动列表（表格布局） | `ob_hudong_list.removeAllViews()` + `takeobacts(str,1)` |
| `009` | 互动列表2 | `ob_hudong2_list.removeAllViews()` + `takeobacts(str,2)` |
| `[2;37;0m` | 重置文本样式 | 清空所有 span 状态 |

## 七、文本渲染（takespan / spantxt）

位于 mudmaind.java 约 941-1150 行：

- 解析 ANSI 风格转义序列 `[...m`
- 支持：前景色、背景色、加粗、超链接、URLSpan、AbsoluteSizeSpan
- `[2;37;0m` = 完全重置格式
- 146 色调色板在 `colors.xml`
- 日/夜/mud 三套配色方案（SharedPreferences `mode` 字段）
